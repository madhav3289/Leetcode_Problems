# Leetcode Problems

This repository contains my solutions to various **LeetCode** problems, primarily written in **Java**.  
Each subfolder corresponds to one problem, including both the problem and code.

---

## Contents

- Solutions organized by problem title or ID.  
- Includes a mix of **Easy**, **Medium**, and **Hard** problems.  
- Covers different topics like:
  - Arrays and Strings  
  - Linked Lists  
  - Stacks and Queues  
  - Trees and Graphs  
  - Dynamic Programming  
  - Backtracking  
  - Greedy and Sliding Window  

---

## Goal

To practice and improve **Data Structures and Algorithms (DSA)** by solving daily LeetCode problems and maintaining clean, readable solutions.
